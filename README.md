# mi_adblock
此项目是自用的网页广告屏蔽规则。

项目地址: https://github.com/m1mark/mi_adblock

# 使用方法
若要使用此规则，请下载支持abp规则的过滤插件（浏览器）。

"!"符号标记的规则不生效，如果需要请自行删除"!"和后面的空格。

"//"符号标记的是简单说明下一行规则生效的对象。

订阅地址: https://raw.githubusercontent.com/m1mark/mi_adblock/master/mi_rules.txt
